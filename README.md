# how-to-ADCS
I created this repo because I want to become more proficient in coding in general but especially ADCS/GNC engineering in particular. I am following "Fundamentals of Spacecraft Attitude Determination and Control" by F. Landis Markley and John L. Crassidis, which I heard multiple times already being referred to as the "ADCS bible". My goal is to simulate a spacecraft tumbling in orbit, model its sensors, actuators, environment and disturbances, and detumble it. In essence, I want to write a simulator, which I can verify against established sources.
